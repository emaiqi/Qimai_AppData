## :dolphin:介绍
### 该项目为爬取[七麦数据](https://www.qimai.cn/rank)平台APP榜单数据
- **20180716 - 七麦JS加密更新,代码也已同步更新,大家放心食用哟!~**
- **20181112 - 增加七麦数据的登录代码(qimai_login.py)，因为有小伙伴反映有些数据(例如关键字排名搜索需要用户登录才可以)**
- **爬虫文件:Spiders目录下的qimai.py**
- **JS解密文件:Spiders目录下的encrypt.js**
